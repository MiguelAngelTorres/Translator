Introducción

Este es un proyecto desarrollado con el fin de simplificar la búsqueda de traducciones de un idioma a otro. Esta pensado para un uso personal y permite la versatilidad que un cuaderno común no posee, ya que permite buscar palabras y sus posibles traducciones a otro idioma de manera rápida y sencilla.
